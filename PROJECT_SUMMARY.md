# 项目总结 - 比特币投资策略回测工具

## 🎯 项目概述

本项目是一个**完全静态的、基于浏览器的**比特币投资策略回测分析工具。它使用 2013-2025 年真实的 BTC 价格数据，帮助投资者：

1. ✅ 对比多种投资策略的历史表现
2. ✅ 分析自定义收益目标的可行性
3. ✅ 通过数据可视化理解市场规律
4. ✅ 做出更理性的投资决策

## 💎 核心功能

### 1. 六种投资策略回测

| 策略 | 类型 | 适用场景 | 交易频率 |
|-----|------|---------|---------|
| HODL | 被动持有 | 长期看好 | 1次 |
| DCA 定投 | 被动定投 | 平滑风险 | 每月1次 |
| 高抛策略 | 主动交易 | 波动市场 | 中等 |
| 低吸策略 | 主动交易 | 下跌市场 | 中等 |
| 网格交易 | 主动交易 | 震荡市场 | 频繁 |
| 均线交叉 | 技术分析 | 趋势市场 | 中等 |

### 2. 自定义收益目标分析

- 投资年限：1-10 年可调
- 目标收益率：10%-100% 可调
- 可行性评估：高/中/低
- 智能建议：基于历史数据

### 3. 数据可视化

- 📈 多策略资产增长曲线对比
- 💹 BTC 历史价格走势（对数坐标）
- 🎯 交互式图表（悬停显示详情）

### 4. 详细数据展示

每个策略显示：
- 初始投资金额
- 最终资产价值
- 收益金额和收益率
- 持有 BTC 数量
- 交易次数
- 策略描述
